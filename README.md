# 📊 Employee Leave Analytics Dashboard

The **Employee Leave Analytics Dashboard** is an interactive business intelligence solution developed using **Power BI** to analyze employee leave data for **FY 2025–26**.

---

## 📌 2. Project Description

This project focuses on analyzing employee leave data to understand **leave trends**, **location-based patterns**, and **employee behavior**.

The goal is to transform raw HR data into meaningful insights that support **better workforce planning**, **leave management**, and **data-driven decision-making**.

---

## ⚙️ 3. Tech Stack

The dashboard was built using the following tools and technologies:

- 📊 **Power BI Desktop** – Dashboard development & visualization  
- 📂 **Power Query** – Data cleaning and transformation  
- 🧠 **DAX** – Calculated measures and business logic  
- 🗃️ **File Format** – `.pbix` (development), `.png` (dashboard previews)  
- 📝 **Data Modeling** – Relationships between tables for cross-filtering  

---

## 📊 4. Data Source

- Internal **Cipla HR Leave Dataset** for **FY 2025–26**  
- Contains **68,000+ records**  
- Fields include: Employee ID, Leave Days, Leave Type, Site, Function, Dates  
- Data cleaned and transformed using **Power Query**  
- Additional fields created:
  - Duration Bucket  
  - Day of Week  
  - Holiday Adjacency  
- Used for analyzing trends, patterns, and employee behavior  

---

## 🚀 5. Features

- 📌 KPI Cards – Total Leave Days & Average Leave per Employee  
- 📈 Trend Analysis – Monthly leave patterns  
- 📍 Location Insights – Site & function-wise comparison  
- 🔥 Top N Analysis – Identify high-leave locations  
- 📊 Heatmap – Site vs Function analysis  
- 📉 Spike Detection – Average reference lines  
- 🧠 Behavior Analysis – Holiday adjacency & weekday patterns  
- 🎛️ Slicers – Dynamic filtering  

---

## 🎯 6. Highlights

### Problem
Employee leave data is large and complex, making it difficult to identify trends and patterns.

### Goal
To analyze leave data and extract insights like trends, hotspots, and behavior patterns.

### Solution
Built an interactive **Power BI dashboard** using:
- Data cleaning (Power Query)
- DAX calculations
- Data modeling

Used visuals like:
- KPI Cards  
- Line Charts  
- Bar Charts  
- Heatmaps  
- Donut Charts  

### Impact
Converted raw data into **actionable insights** for better workforce planning.
